# gg-excel-data-analysis-projects
Portfolio of Excel data analysis projects with interactive dashboards,pivot tables,and KPI visualizations.
Penguins Data Analysis Dashboard

# Objective
The project analyzes penguin species data to understand differences in physical characteristics such as body mass, flipper length, and overall size classification.

# Tools Used
- Microsoft Excel (Pivot Tables, Slicers, Dashboard)
- Data Cleaning & Lookup Functions (VLOOKUP, IF)

# Dataset
The dataset includes:
- Species (Adelie, Chinstrap, Gentoo)
- Island location
- Body Mass (g)
- Flipper Length (mm)
- Culmen Length & Depth
- Sex

# Key Insights
- Gentoo penguins are the largest, with significantly higher body mass and flipper length
- Flipper length is the best variable for distinguishing between species
- Male penguins tend to have higher measurements than females
- The dataset shows imbalance across species distribution
- Higher body mass values are concentrated in specific species (right-skewed distribution)

# Dashboard Features
- KPI cards (Avg Body Mass, Heaviest Penguin, Avg Flipper Length, % >5000g)
- Interactive slicers (Species, Island, Sex)
- Charts comparing species characteristics and distributions
- Clean one-page executive dashboard
